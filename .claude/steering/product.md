# Product Steering

## Product Vision
A Personal AI Employee that operates as a digital FTE (Full-Time Equivalent), managing personal and business affairs 24/7 through autonomous agents.

## Target Users
- Busy professionals seeking automation
- Small business owners needing operational support
- Developers building AI agent systems
- Panaversity hackathon participants

## Core Value Propositions
1. **24/7 Operation** - Continuous monitoring and action
2. **Autonomous Execution** - Minimal human intervention required
3. **Human-in-the-Loop** - Approval workflow for sensitive actions
4. **Local-First Privacy** - Data stays on user's machine
5. **Extensible Architecture** - MCP-based integrations

## Product Principles

### User Experience
- **Transparent Operations** - Users always know what the agent is doing
- **Graceful Degradation** - Clear error messages and recovery paths
- **Progressive Disclosure** - Start simple, reveal complexity as needed
- **Audit Trail** - All actions logged for review

### Feature Prioritization (MoSCoW)
- **Must Have**: Watchers, Claude Code integration, approval workflow, logging
- **Should Have**: Multiple watchers, email MCP, plan generation
- **Could Have**: Odoo integration, social media MCPs, CEO briefings
- **Won't Have (Now)**: Cloud sync, multi-agent coordination

## Success Metrics
- Tasks completed autonomously per day
- Human intervention rate
- Error recovery success rate
- User satisfaction score

## Release Strategy
- Bronze: Foundation (watcher + Claude Code + vault)
- Silver: Functional Assistant (multiple watchers + approval workflow)
- Gold: Autonomous Employee (full MCP suite + briefings)
- Platinum: Cloud + Local (distributed operation)

## Competitive Advantages
- Ralph Wiggum Loop persistence pattern
- Claim-by-Move task ownership
- File-based human-in-the-loop
- Obsidian as memory/GUI
