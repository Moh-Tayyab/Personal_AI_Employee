# Generate Briefing - Agent Skill

Creates comprehensive CEO briefing reports with business intelligence, progress metrics, and actionable recommendations for decision-making.

## When to Use

- Weekly business review (Monday mornings)
- User commands: `/generate-briefing`, "briefing", "weekly summary", "status report"
- On-demand status check
- Before important meetings

## Before Implementation

| Source | Gather |
|--------|--------|
| **Done/** | Completed tasks this period |
| **Needs_Action/** | Pending items requiring attention |
| **Pending_Approval/** | Items awaiting decisions |
| **Plans/** | Active and completed plans |
| **Business_Goals.md** | Targets, KPIs, priorities |
| **Logs/** | Activity metrics |

## Workflow

### Phase 1: Data Collection
1. Scan `Done/` for completed tasks (last 7 days)
2. Check `Needs_Action/` for pending items
3. Review `Pending_Approval/` for blocked decisions
4. Analyze `Plans/` for active initiatives
5. Read `Business_Goals.md` for targets

### Phase 2: Analysis
Calculate metrics:
- Tasks completed vs. target
- Average response time
- Approval backlog
- Bottleneck identification
- Progress against goals

### Phase 3: Intelligence Generation
1. Identify patterns and trends
2. Flag urgent items requiring attention
3. Generate proactive suggestions
4. Highlight upcoming deadlines

### Phase 4: Report Creation
Write briefing to `Briefings/BRIEFING_YYYYMMDD.md`

## Metric Calculations

### Response Time
```python
def calculate_avg_response_time(completed_tasks):
    """Calculate average response time in hours."""
    times = []
    for task in completed_tasks:
        if task.get('received_at') and task.get('completed_at'):
            delta = task['completed_at'] - task['received_at']
            times.append(delta.total_seconds() / 3600)
    return sum(times) / len(times) if times else 0
```

### Completion Rate
```python
def calculate_completion_rate(done_count, total_count):
    """Calculate task completion rate."""
    return (done_count / total_count * 100) if total_count > 0 else 0
```

### Bottleneck Detection
```python
def detect_bottlenecks(items):
    """Identify process bottlenecks."""
    bottlenecks = []
    for item in items:
        if item.get('waiting_for_approval'):
            bottlenecks.append({
                'type': 'approval',
                'item': item['id'],
                'age_days': item['age_days']
            })
        if item.get('blocked_by'):
            bottlenecks.append({
                'type': 'dependency',
                'item': item['id'],
                'blocked_by': item['blocked_by']
            })
    return bottlenecks
```

## Error Handling

| Error | Recovery Action |
|-------|-----------------|
| No data in Done/ | Report "No completed tasks this period" |
| Business_Goals.md missing | Use default metrics, note in report |
| Cannot calculate metrics | Skip metric, continue with others |
| Cannot write to Briefings/ | Create directory, retry |
